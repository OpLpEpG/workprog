Чтение памяти

Для каждого модуля устройства чтение памяти производится отдельно.

Перед чтением памяти необходимо убедится, что модуль находится (Режим информации) в состоянии "выключен", остановить режим информации.

Окно чтения памяти модуля Рис2,3 открывается из меню: "Чтение памяти модуля..." ,всплывающего меню окна управления устройствами, когда мышь находится над модулем Рис1.

Выбор режима чтения модуля

По последовательному порту Рис3, для снижения времени считывания: выбираем повышенную скорость обмена данными поддерживаемую модулем, можно для увеличения скорости увеличить длину принимаемых пакетов данных, снять  флажок "Создать файл рассчетных данных", что увеличивает скорость чтения данных, но при визуализации считанных данных, будет больше задекжка. Можно задать создание дополнительного бинарного файла.

Память модуля определяется как диск Рис2. Выбираем диск "Диск SD", можно снять  флажок "Создать файл рассчетных данных", что увеличивает скорость чтения данных.

Выбор диапазона данных

Примерный диапазон записаных данных за время работы прибора выделен голубым или синим цветом. При выборе диапазона считываемых данных нельзя  чтобы  левая  граница данных была вне диапазона записаных данных

Можно задать диапазон вручную:  первая колонка – время относительно начала работы прибора , вторая колонка – время по компьютеру

Лучше считать данные за все время работы прибора т.к. вседа можно зкспортировать нужный отрезок.

Считывание

При нажатии на кнопку "Старт" начинается считывание памяти модуля,   если все нормально идет обновление линии прогресса,  области статуса окна, считывание можно прервать вручную кнопка "Прервать".

По окончании считывания закрыть окно кнопка "Выход", затем можно приступать к считыванию памяти другого модуля на линии.
