Экспортирование данных проекта

После считывания памяти всех модулей, для далнейшей обработки данных в программе "Горизонт", необходимо произвести экспорт Рис1.

Рис1

Данные комплекса преобразуются в формат ПСК6("MBK"), данные профилемера в формат "ap_pb" Рис2,3

Рис2 Рис3

Выбор файла

при нажатии на кнопку  Рис2-1 появится стандартное диалоговое окно открытия файла, необходимо ввести имя создаваемого файла (можно без расширения)

Выбор диапазона данных

Лучше задать нужный интервал вручную:  первая колонка – время относительно начала работы прибора или вторая колонка – время по компьютеру

Экспорт

При нажатии на кнопку "Старт" начинается экспортирование данных,   если все нормально идет обновление линии прогресса,  области статуса окна, экспортирование можно прервать вручную кнопка "Прервать".

По окончании экспорта будет создан "if" файл (для профилемера "if, ak1")

Длина ФКД для профилемера 682 не менять Рис3.
